# 📊 EDA Project: Mobile Sales

This project performs **Exploratory Data Analysis (EDA)** on a mobile phone sales dataset. The notebook includes data cleaning, visualization, and insights generation to understand key factors influencing mobile phone sales.

## 🔍 Key Highlights:

* Data preprocessing and handling missing values
* Univariate and multivariate analysis
* Correlation heatmaps and feature importance
* Insights on price trends, brand performance, and feature impact

## 🛠 Tools & Libraries:

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
